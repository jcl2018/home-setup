# D10: Docs Have Clear Lanes

Each documentation file has one job.

Key assignments:
- DECISIONS.md: decision journal entries
- CLAUDE.md: layout tree, rules, machine config
- README.md: orientation, links to doc tiers
- SKILLS-CHEATSHEET.md: skill commands to type
- COMMANDS-AND-RULES.md: project commands and rules
- INSPECTION-BASELINE.md: known-good state

Do not duplicate content across docs. If two docs cover the same topic,
one will drift.
