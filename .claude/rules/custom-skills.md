# P3: Custom Skills Are Ours

Skills in .claude/skills/ are authored in this repo. When creating or modifying:
1. Add or update the entry in skills-catalog.json with "source": "custom"
2. Update docs/operations/SKILLS-CHEATSHEET.md
3. Run bash scripts/deploy.sh after changes

See CLAUDE.md Principles section for the full rationale.
